[内部文件] 深南市劳动局关于落实1997年7月国有企业下岗职工基本生活保障和再就业工作的实施细则

发文单位：深南市劳动局
发文日期：1997年7月15日
文件编号：劳发〔1997〕第078号
国有企业改革
职工安置
再就业工程

一、总则

为贯彻落实国家关于做好国有企业下岗职工基本生活保障和再就业工作的指示精神，结合我市实际情况，制定本实施细则。本细则旨在规范下岗职工的基本生活保障，促进其再就业，维护社会稳定，保障我市经济体制改革的顺利进行。

各国有企业、相关政府部门和社会服务机构应按照本细则要求，切实履行责任，共同做好下岗职工的基本生活保障和再就业工作。对于因企业结构调整、生产经营困难等原因下岗的职工，应确保其基本生活，并通过多种渠道促进其再就业。

二、基本生活保障措施

1. 各国有企业应设立下岗职工基本生活保障专项资金，确保按时足额发放基本生活费。基本生活费标准不得低于当地城镇居民最低生活保障线。

2. 对于特别困难的下岗职工家庭，经审核符合条件的，可适当提高补助标准或提供临时救助。

3. 下岗职工的社会保险关系应予以保留，企业应继续为其缴纳养老保险和医疗保险费用，确保其社会保险权益不受影响。

4. 各级劳动部门应建立下岗职工动态管理档案，及时掌握下岗职工的基本情况和再就业需求。

三、再就业工作安排

1. 实施"再就业工程"，通过职业介绍、职业培训、创业扶持等多种方式，促进下岗职工尽快实现再就业。

2. 各行业主管部门应积极协调本系统内企业，优先招用符合条件的下岗职工。对于吸纳下岗职工达到一定比例的企业，可给予相应的政策优惠。

3. 加强职业培训工作，特别是针对市场需求较大的技能培训。在春季扩招期间，各职业培训机构应增加培训名额，优先面向下岗职工开展免费或低收费培训。

4. 鼓励下岗职工自主创业，提供创业指导、小额贷款等扶持政策。对于成功创业并吸纳其他下岗职工就业的，给予额外奖励。

5. 加强与用人单位联系，收集招工信息，定期举办针对下岗职工的专场招聘会。对于大规模招工的企业，劳动部门应主动对接，推荐合适人选。

6. 在1997年下半年的春季扩招计划中，各行业应预留不少于30%的岗位用于吸纳下岗职工。特别是服务行业、社区服务等领域，应加大招工力度。

四、组织与实施

1. 市劳动局成立下岗职工基本生活保障和再就业工作领导小组，负责统筹协调全市相关工作。

2. 各区县劳动部门应成立相应工作机构，制定具体实施方案，确保各项政策措施落到实处。

3. 建立定期报告制度，各国有企业每月向所属劳动部门报告下岗职工基本生活保障和再就业工作情况。

4. 加强监督检查，对于不执行本细则规定、挪用专项资金、侵害下岗职工权益的单位和个人，依法严肃处理。

五、附则

1. 本细则自发布之日起施行。

2. 本细则由深南市劳动局负责解释。

3. 各区县、各国有企业可根据本细则制定具体实施办法。

深南市劳动局办公室 1997年7月印发
